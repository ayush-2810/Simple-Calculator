This is a simple calculator made in python using **tkinter** library.


  Symbol   | Operations 
|:---:|:---:| 
|    +     |  Addition            |
|    -     |  Subtraction         |
|    x     |  Multiplication      |
|    /     |  Division            |
|   xx     |  Power               |
|   Del    |  Delete last integer |
|    C     |  Clear screen        |
|    =     |  Equal               | 
|    .     |  Decimal             |
